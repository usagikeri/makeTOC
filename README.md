# daimoku__generator
題目を自動生成するスクリプト．

# Discription
題目を自動生成するスクリプト．

# Install

```
$ git clone
```

# Requirement

- pyyaml

```
$ pip install pyyaml
```

# Usage
member.yamlに学籍番号，名前，タイトルを記述する．

```
B3:
  - [学籍番号,名前,タイトル]
B4:
  - [学籍番号,名前,タイトル]
M1:
  - [学籍番号,名前,タイトル]
```

# Run

```
$python daimokugen.py
```
